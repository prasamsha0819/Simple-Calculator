# Simple-Calculator
JavaFX calculator class project
<br>

## Features
- Add and subtract numbers
- Clear the display
- Handles multiple consecutive operations
- User-friendly GUI with styled buttons and display

## How to Run
1. Make sure you have **Java** and **JavaFX** installed on your computer.
2. Navigate to the folder containing `SimpleCalculator.java`.
3. Compile the program:
   ```bash
   javac SimpleCalculator.java
